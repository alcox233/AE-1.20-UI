# AE-1.20-UI Multi-module adaptation

The AE-UI texture pack for **Minecraft 1.20** includes certain text and texture corrections.

We have added AE-style UIs for a large number of mods, available in both light and dark versions, such as:
- **Farmer's Delight**
- **Sophisticated Storage / Core**
- **Avaritia**
- ……UIs for other mods are currently in the pipeline.

## 📌 Note
Specifically adapted for the [GTO (GregTech-Odyssey) Modpack](https://gtodyssey.com/).

## ❓ Q&A
**Q: How to use this texture pack?**  
A: Place the resource pack at the top of the resource pack interface. It has undergone many modifications and is not compatible with other AE UI resource packs.

## 📦 Versions
- Supports Minecraft 1.20
- Light & Dark variants included
