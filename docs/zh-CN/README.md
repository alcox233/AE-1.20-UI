# AE-1.20-UI 多模组适配

适用于 **1.20版本** 的 AE-UI 材质包，包含一定的文本修正和贴图修正。

为大量模组添加了 AE 风格的 UI，分亮色和暗色 2 个版本，例如：
- **农夫乐事** (Farmer's Delight)
- **精妙存储** (Sophisticated Storage/Core)
- **无尽贪婪** (Avaritia)
- ……其他模组的 UI 正在加紧制作中。

## 📌 注
特别针对 [GTO 整合包](https://gtodyssey.com/zh-hans/) 进行适配。

## ❓ Q&A
**Q: 如何使用本材质包？**  
A: 将材质包放在资源包界面最上面。本包修改了大量内容，与其他 AE 风格的 UI 材质包不兼容。
