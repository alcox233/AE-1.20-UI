# English

- source.txt: Paths of images to process (can be folders). It is recommended to use absolute paths. Each line is a path.
- backup.sh: Temporarily store the original data
- replace.sh: Perform pixel replacement
- clean.sh: Clean up the original data
- once.sh: One-click processing script
